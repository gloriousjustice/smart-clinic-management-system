# Smart Clinic Management System User Stories

## Admin User Stories
* As an Admin, I want to log into the secure admin portal so that I can manage system settings.
* As an Admin, I want to add new doctors to the system with their specialization and credentials.

## Doctor User Stories
* As a Doctor, I want to log into my portal so that I can view my daily schedule.
* As a Doctor, I want to view a list of all my scheduled patient appointments.

## Patient User Stories
* As a Patient, I want to register and log into the system to manage my health appointments.
* As a Patient, I want to search for doctors by their name or specialization to find the right care.
